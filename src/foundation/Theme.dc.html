How the switch works
The mechanism behind the bottom-right toggle.
  • Local storage: preference persisted via localStorage('theme') — light / dark / system.
  • Root attribute: chrome.js sets data-theme="dark" on <html>.
  • CSS re-binds: the [data-theme="dark"] block in colors_and_type.css re-points every --color-* token — no component or screen is touched.
